# Prerequisites 

You need to have the following commands installed: *catenate*, *fs*, *odbc*, *replacestrvar*, *rename*, *sdecode*, *sxpose* and *tosql*.

For example, to install *fs* you must run this line of code:
 
      ssc install fs
